Ok so by default this is trying to stream the new president video. If you want it to play something else, you have to modify the "source" query string to point to the file on akamai on the end of the URL paths to the SWF file. Look at the embed code for the player, find where it says akamai_od_player.swf (there should be a few places) and modify ?source=whatever.mp4 to the end of that, before the closing quotation marks.

this player only supports playback of h264 files, not vp6 flv files.
