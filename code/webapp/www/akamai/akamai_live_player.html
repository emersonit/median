Akamai Live Stream Players

There are two versions of the Akamai Live Player: one version that features buttons/controls that are accessible when the user is in full screen mode, and another version that removes the full screen controls, requiring the user to press the ESCAPE key to leave full screen mode.

It is strongly advised that any public use of the Akamai Stream player uses the version that has the fullscreen controls. The second version is only to be used for special circumstances when the live video must take over the screen, such as for a kiosk.
